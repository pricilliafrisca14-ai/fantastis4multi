website Kelompok 2

Membangun website dari nol dengan HTML, CSS, dan struktur responsif.

tentang Desain Pengkodean Pengujian

1. tentang

kami adalah kelompok 2,sebuah tim mahasiswa yang berfokus pada pengembangan dan pembelajaran website berbasis html dan css. website ini dibuat sebagai media panduan pembelajaran yang bertujuan untuk membantu pengguna memahami dasar-dasar perancangan dan pembuatan website secara sderhana,terstruktur,dan mudah di pahami.

Gambar: Contoh wireframe untuk halaman utama, menunjukkan header, navigasi, dan area konten.

2. Kodingan

Pada tahap pengkodean,website ini dibangun menggunakan bahasa pemograman html dan css.html digunakan untuk menyusun stuktur dasar halaman website, seperti header,menu nagivasi,konten,dan footer.sementara itu,css digunakan untuk mengatur tampilan visual agar website terlihat lwbih rapi,menarik,dan mudah

Gambar: Mockup desain dengan palet warna dan elemen UI seperti tombol dan kartu.

3. Pengkodean

pengkodean merupakan tahap penerapan hasil perancangan website ke dalam bentuk bahasa pemograman.pada tahap ini,pengembangan website dilakukan dengan menggunakan html sebagai penyusun struktur halaman,css untuk mengatur tampilan visual,serta javascript untuk memberikan fungsi interaktif sederhana.

  • Header: sebagai bagian identitas website yang menampilkan judul dan elemen pendukung lainnya.
  • Navigasi: yang berfungsi sebagai menu penghubung antar halaman website.
  • Main: sebagai area utama untuk menampilkan isi dan informasi panduan.
  • Footer: yang berisi informasi tambahan seperti keterangan kelompok dan hak cipta.
Gambar: Contoh kode HTML dengan elemen dasar dan gaya CSS.

4. pembuatan QR jasa

pembuatan QR code dilakukan untuk memudahkan akses ke layanan atau halaman website.QR code dibuat dari tautan website dan diuji agar dapat dipindai dengan baik menggunakan berbagai perangkat.QR code ini membantu pengguna mengakses informasi secara cepat dan praktis.

Gambar: Screenshot pengujian responsivitas di desktop dan mobile.
